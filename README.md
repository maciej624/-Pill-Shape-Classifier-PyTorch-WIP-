# 💊 Pill Shape Classifier (PyTorch) 🚧 [WIP]

> **Note:** This project is currently under active development (Work in Progress).

## 📖 About The Project
This project focuses on building a computer vision system capable of automatically recognizing and classifying different pill shapes from images. The core classification engine is built upon the **ResNet18** convolutional neural network architecture, implemented using PyTorch.

The goal is to bridge the gap between classic industrial automation and modern AI by providing a reliable model for quality control and sorting processes.

## 🛠️ Built With
* **Language:** Python 3.x
* **Deep Learning Framework:** PyTorch, Torchvision
* **Computer Vision:** OpenCV / PIL
* **Model Architecture:** ResNet18 (Transfer Learning)
